¡Hola! Te damos la bienvenida a Articlor

Una herramienta diseñada para facilitar la creación y gestión de tus escritos.

Con esta aplicación, puedes expresar tus ideas, investigaciones o historias de una manera organizada y eficiente.

  • Version 1.0 Release:
  • Escribir y formatear artículos de manera intuitiva.
  • Guardar y abrir artículos en diferentes formatos.
  • Interfaz fácil de usar con acceso rápido a todas las funciones.
  • Personalizar la apariencia del editor según tus preferencias.
  • Exportar artículos en diversos formatos, como texto plano o HTML.
  • Compatibilidad con sistema operativo Windows y Mobile mediante navegadores web.
